LazyNvim

在 Windows 10 上安装 LazyNvim 的步骤如下：

1. 安装 Neovim

LazyNvim 是基于 Neovim 的配置框架，因此需要先安装 Neovim。

  1. 下载 Neovim：

    • 访问 Neovim 官方 GitHub Releases 页面。

    • 下载适合 Windows 的最新版本（例如 nvim-win64.zip 或 nvim-win32.zip）。

    • 解压文件并将解压后的 nvim.exe 文件放到一个目录中（例如 C:\Program Files\Neovim）。

  2. 配置环境变量：

    • 打开系统环境变量设置，将 Neovim 的安装路径（例如 C:\Program Files\Neovim）添加到 PATH 环境变量中。

    • 重启终端或命令提示符，输入 nvim，如果能正常启动 Neovim，则说明安装成功。

2. 安装 Git

LazyNvim 需要通过 Git 克隆仓库，因此需要安装 Git。

  1. 下载并安装 Git：

    • 访问 Git for Windows 官方页面。

    • 下载并安装 Git。

  2. 验证安装：

    • 打开终端，输入 git --version，如果显示 Git 版本号，则说明安装成功。

      这会将 LazyNvim 的配置文件克隆到 Neovim 的配置目录中。

  2. 启动 Neovim：

    • 在终端中输入 nvim 启动 Neovim。

    • 第一次启动时，LazyNvim 会自动安装所需的插件。

  3. 验证安装：

    • 如果安装成功，Neovim 会显示 LazyNvim 的欢迎界面。

5. 配置 LazyNvim

LazyNvim 提供了默认配置，你可以在 ~/.config/nvim/lua/config 目录下修改配置文件以满足个人需求。

  1. 修改配置文件：

    • 打开 ~/.config/nvim/lua/config/options.lua 文件，修改编辑器选项。

    • 打开 ~/.config/nvim/lua/plugins 目录，添加或修改插件配置。

  2. 安装额外插件：

    • 在 ~/.config/nvim/lua/plugins 目录下添加插件配置，例如：

      lua复制

  3. 重启 Neovim：

    • 保存配置文件后，重启 Neovim，LazyNvim 会自动安装新配置的插件。

    • 确保 Neovim 的安装路径已添加到系统环境变量 PATH 中。

    • 确保 Git 已正确安装。

  • 如果插件未自动安装：

    • 确保 ~/.config/nvim/init.lua 文件中包含以下内容：

  • 如果需要手动安装插件：

    • 在 Neovim 中输入以下命令：
